# Recolector de Monedas
## Descripción del Juego
**Recolector de Monedas** es un juego en Unity donde el jugador debe recolectar 13 monedas distribuidas a lo largo de un mundo interactivo. El jugador se mueve haciendo click con el ratón, navegando a través de rampas, escaleras, y utilizando teletransportes. Todos los datos del juego, incluyendo las monedas recolectadas y la información del jugador, se almacenan en una base de datos.

## Características
- **Movimiento del Jugador:** El jugador se mueve haciendo click en el ratón.
- **Recolecta de Monedas:** Recolecta las 13 monedas distribuidas por el mundo.
- **Entorno Interactivo:** Incluye rampas, escaleras y teletransportes para un juego dinámico.
- **Base de Datos:** Los datos del juego, como las monedas recolectadas y la información del jugador, se almacenan en una base de datos SQLite.
